5.1. MicroPython 语法

MicroPython旨在实现了Python 3.4标准，
而且大部分MicroPython的特征跟Python是相同的, Python语法请参阅
`docs.python.org <https://docs.python.org/3/reference/index.html>`_.

MicroPython的官方文档请参阅
`docs.micropython.org <https://docs.micropython.org/en/latest/>`_.
