5.2. API库
========================

Python 的强大很大程度上归功于其丰富的开源库生态，这些库覆盖了从基础数据处理到前沿人工智能的各个领域，内置的标准库内置功能覆盖文件操作、网络编程、数据解析等，
还有第三方的库生态，如：


  * 数据科学：NumPy、Pandas、Matplotlib

  * 机器学习：Scikit-learn、TensorFlow、PyTorch
  
  * Web 开发：Django、Flask、FastAPI

用户通过引用这些库，实现丰富的应用场景，以下是标准库、Micropython、 掌控板固件库相关内容。
